# OneStep 사용 설명서

## 1. 다운로드와 실행

1. GitHub 릴리즈 페이지에서 `OneStep-Windows_v1.0.zip`을 다운로드합니다.
2. 파일 무결성 확인이 필요하면 SHA256 해시를 확인합니다.
3. ZIP 파일의 압축을 풉니다.
4. `OneStep_v1.0.exe`를 실행합니다.

Windows SmartScreen 경고가 표시되면 [Windows SmartScreen 안내](smartscreen.md)를 확인하세요.

## 2. 파일 추가

메인 목록 영역에 파일이나 폴더를 드롭합니다.

지원 입력:

- ZIP
- PDF
- TIFF
- JPG, JPEG, PNG, BMP
- HWP

새 파일을 드롭하면 기존 목록은 새 입력 목록으로 교체됩니다.

## 3. 처리 방식

OneStep은 모든 파일을 로컬 PC에서 처리합니다.

- ZIP 파일은 압축을 풉니다.
- PDF 파일은 페이지별 이미지로 분해합니다.
- TIFF 파일은 프레임별 이미지로 분해합니다.
- HWP 파일은 로컬 PC에 설치된 한글 프로그램으로 PDF로 변환합니다.
- 이미지는 설정에 따라 회전, 리사이징, PNG-JPG 변환을 수행합니다.

## 4. 미리보기

목록에서 하나 이상의 항목을 선택하면 우측 미리보기 패널이 열립니다.

- 단일 이미지: 큰 미리보기
- 여러 이미지: 그리드 미리보기
- 여러 페이지 PDF: 그리드에서는 첫 페이지와 페이지 수 표시

목록 항목을 더블클릭하면 Windows 기본 앱으로 파일을 엽니다.

## 5. 이미지 편집창

미리보기 이미지를 클릭하면 이미지 편집창이 열립니다.

1. 남길 영역을 사각형으로 드래그합니다.
2. 저장하면 현재 이미지가 선택 영역으로 잘립니다.
3. 이미지 편집창의 뒤로/앞으로 버튼으로 편집 이력을 이동할 수 있습니다.
4. 각도 보정이 필요하면 기울임 모드를 사용합니다.

이미지 편집창에서는 영역 자르기, 회전, 기울임 보정, 다른 이름으로 저장을 사용할 수 있습니다.

이미지 편집 이력은 로컬 임시 폴더에 저장됩니다.

## 6. PDF 도구

### PDF 변환

이미지 파일을 선택한 뒤 `PDF 변환`을 누르면 각 이미지가 개별 PDF 파일로 변환됩니다.

### PDF 묶음

여러 이미지를 선택한 뒤 `PDF 묶음`을 누르면 하나의 PDF가 생성됩니다. 페이지 순서는 목록 순서를 따르며, 같은 PDF에서 분해된 이미지라면 페이지 메타데이터를 우선 사용합니다.

## 7. 설정

설정할 수 있는 항목:

- 임시 폴더
- 회전
- 리사이징
- PNG-JPG 변환
- PDF 변환 후 원본 삭제
- PDF 묶음 후 원본 삭제
- 압축파일 작업 후 원본 삭제
- 압축파일을 현재 위치에 풀기
- 긴 변 최대값
- JPEG 품질

## 8. 임시 폴더

기본 임시 폴더는 앱 폴더 아래 `data/temp`입니다. 임시 폴더에는 작업 중 필요한 로컬 작업 파일만 저장됩니다.

OneStep이 종료된 상태에서는 사용자가 임시 폴더를 삭제해도 됩니다.
