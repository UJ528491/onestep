# 서드파티 고지

OneStep은 다음 서드파티 패키지와 Windows 플랫폼 API를 사용합니다.

## 런타임 의존성

- PySide6 / Qt for Python: LGPL-3.0-only 또는 GPL-2.0-only 또는 GPL-3.0-only
- PyInstaller: 부트로더 예외가 포함된 GPL-2.0-or-later
- Pillow: HPND 라이선스
- OpenCV Python headless: Apache-2.0
- NumPy: BSD-3-Clause
- pywin32: PSF 스타일 라이선스
- winrt-Windows.* 패키지: MIT

## 플랫폼 API

- Windows 런타임 OCR
- Windows 런타임 PDF
- HWP 변환용 Windows COM 자동화

이 문서는 안내용입니다. 정확한 라이선스 전문은 각 패키지의 공식 라이선스 메타데이터와 저장소를 확인하세요.
