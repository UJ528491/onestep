# 개인정보 처리 안내

OneStep은 Windows PC에서 실행되는 로컬 전용 문서 처리 도구입니다. 작업 파일, OCR 결과, 파일명, 파일 경로, 설정값, 오류 정보, 사용 통계를 외부 서버로 전송하지 않습니다.

## 수집하지 않는 정보

OneStep은 다음 정보를 수집하거나 전송하지 않습니다.

- 이름, 전화번호, 이메일, 계좌번호, 주민등록번호 같은 개인정보
- 고객사명, 거래처명, 내부 URL, IP 같은 업무 정보
- 파일명, 시트명, 전체 로컬 파일 경로
- 문서 내용, OCR 원문, 파싱 결과
- 사용자 계정명, PC명
- 토큰, 쿠키, API 키, 인증정보
- 스크린샷
- 사용 통계, 클릭 기록, 텔레메트리
- 오류 로그, 작업 로그

## 로컬에 저장될 수 있는 파일

OneStep이 로컬 PC에 저장할 수 있는 파일은 다음 범위입니다.

- `data/settings.json`: 기능 켜기/끄기, 임시 폴더 경로, 리사이징 크기, JPG 품질 같은 앱 설정값
- `data/temp`: 작업 중 필요한 원본 사본, PDF/TIFF 분해 이미지, 미리보기 이미지, 이미지 편집 이력, PDF 묶음 작업용 임시 파일
- 사용자가 선택한 작업 폴더 또는 원래 파일 위치의 결과물

`data/settings.json`에는 문서 내용, OCR 텍스트, 파일명 목록, 전체 파일 경로, 계정 정보, 인증정보가 저장되지 않습니다.

## 로그 정책

현재 릴리즈는 런타임 로그 파일을 생성하지 않습니다. 별도의 로그 폴더도 만들지 않습니다.

작업 상태와 오류 메시지는 앱 화면에만 표시됩니다. 앱 종료 후 OCR 원문, 문서 내용, 파일명, 전체 파일 경로, 오류 정보를 로그 파일로 남기지 않습니다.

## 임시 폴더

임시 폴더는 문서 처리를 위해 필요한 로컬 작업 공간입니다. 새 드롭 작업이 시작되면 앱은 설정된 임시 폴더를 비웁니다.

강제 종료, 전원 차단, OS 오류 같은 비정상 종료 상황에서는 임시 폴더에 작업 중간 파일이 남을 수 있습니다. 이 파일은 외부로 전송되지 않으며, 필요하면 사용자가 삭제할 수 있습니다.

## OCR 및 HWP 처리

OCR은 Windows 내장 OCR 엔진을 로컬에서 호출합니다. HWP 변환은 사용자의 PC에 설치된 한글 프로그램을 Windows COM 자동화로 제어합니다.

두 기능 모두 클라우드 OCR, 외부 API, 외부 변환 서버를 사용하지 않습니다.
