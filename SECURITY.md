# 보안 안내

## 로컬 전용 실행

OneStep은 사용자의 Windows PC 안에서 파일을 처리하는 로컬 전용 앱입니다.

- 외부 서버 업로드 기능 없음
- 클라우드 OCR 또는 외부 API 호출 없음
- 텔레메트리, 분석 SDK, 사용 통계 수집 없음
- 런타임 로그 파일 생성 없음
- 토큰, 쿠키, API 키 같은 인증정보 저장 없음

## Windows SmartScreen

현재 공개 빌드는 코드 서명 인증서가 없는 PyInstaller 실행 파일입니다. 따라서 Windows SmartScreen에서 `Windows의 PC 보호` 같은 파란색 경고가 표시될 수 있습니다.

이 경고는 파일의 게시자 평판과 코드 서명 여부에 대한 경고입니다. 앱이 외부로 데이터를 전송한다는 의미는 아닙니다.

확인 방법과 실행 절차는 [Windows SmartScreen 안내](docs/smartscreen.md)를 참고하세요.

공개 릴리즈 파일은 GitHub Actions Windows runner에서 테스트와 빌드를 거친 뒤 생성됩니다. 릴리즈 페이지에는 ZIP 파일과 SHA256 파일이 함께 첨부됩니다.

## 로컬 파일 보안 범위

앱은 작업을 위해 로컬 임시 폴더를 사용합니다. 임시 폴더에는 작업 중 필요한 원본 사본, PDF/TIFF 분해 이미지, 미리보기 이미지, 이미지 편집 이력 같은 파일이 저장될 수 있습니다.

새 드롭 작업이 시작되면 임시 폴더를 비웁니다. 다만 앱 강제 종료, 전원 차단, OS 오류가 발생하면 임시 폴더에 작업 중간 파일이 남을 수 있습니다.

## 파일 삭제 동작

설정에서 원본 삭제 옵션이 켜져 있으면 일부 작업 후 원본 파일을 제거할 수 있습니다. 삭제는 직접 영구 삭제가 아니라 Windows 휴지통 이동 방식을 사용합니다.

휴지통 이동이 OS 권한, 경로 상태, 보안 정책 때문에 실패하면 앱은 작업 결과를 유지하고 삭제를 건너뜁니다.

## HWP 변환

HWP 변환은 사용자의 PC에 설치된 한글 프로그램을 Windows COM 자동화로 제어합니다.

- HWP 파일은 외부 서버로 전송되지 않습니다.
- 한글 프로그램의 보안/권한 확인 창이 필요한 환경에서는 사용자가 허용해야 할 수 있습니다.
- 변환 실패나 비정상 종료 시 임시 폴더에 작업 중간 파일이 남을 수 있습니다.

## 보안 이슈 제보

보안 문제를 제보할 때 실제 업무 문서나 개인정보가 포함된 파일을 첨부하지 마세요.

가능하면 다음 정보만 포함해 주세요.

- OneStep 버전
- Windows 버전
- 관련 파일 형식
- 개인정보를 제거한 재현 절차
- 필요 시 개인정보를 가린 스크린샷
